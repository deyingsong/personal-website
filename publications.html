Tools and Software

  • How task specification affects generalization in machine learning: A tutorial in Neuromatch Academy NeuroAI Course, 2024.
  • Awesome Foundation Models for the Brain: A professionally curated list of Modeling papers, Datasets, Experimental papers, Code Repositories, and Benchmarks for Foundation Model for the Brain.
  • High-Performance EIF Neural Network Simulation: A collection of optimized C implementations for simulating large-scale Exponential Integrate-and-Fire (EIF) neural networks with various connectivity patterns and input modalities.
  • Alpha Signal Discovery: A comprehensive Python framework for alpha signal discovery in financial data, adapted to ~2,000 samples. The framework includes data analysis, model development, validation, backtesting, and risk management.
  • Statistical Arbitrage Backtesting Engine: A C++ backtesting engine for statistical arbitrage and quantitative trading, featuring an event-driven architecture, low-latency execution, statistical validation, and production readiness
  • codes for manuscript: Neuronal heterogeneity of normalization strength in a circuit model, Song, D., Ruff, D., Cohen, M., & Huang, C. (2024), bioRxiv.
